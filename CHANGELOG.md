# Changelog

All notable changes to Savannah Code will be documented here.

## [1.0.0] — Initial Release

### Added
- **Savannah Sunset** dark theme — primary warm dark variant
- **Savannah Dawn** light theme — warm light variant for daytime use
- **Safari Midnight** deep dark theme — high contrast night variant
- Full semantic token support for all three variants
- Language-specific token rules:
  - JavaScript / TypeScript
  - Python (decorators, built-ins)
  - Rust (lifetimes, macros)
  - Go (built-in functions)
  - HTML / CSS / SCSS
  - JSON / YAML
  - Markdown (headings, bold, italic, links, blockquotes, lists)
  - Shell / Bash
- Complete VS Code UI theming:
  - Editor, sidebar, activity bar
  - Status bar, title bar, panel
  - Terminal ANSI colors (all 16)
  - Tab bar, breadcrumbs
  - Git decorations
  - Diff editor colors
  - Minimap, scrollbars
  - Hover widgets, suggest widget
  - Input, buttons, dropdowns, badges
  - Peek view
  - Notification center
- WCAG AA accessibility compliance
- Semantic highlighting enabled for all variants
